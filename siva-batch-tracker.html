⚠️ Configuration requise

Pour utiliser l'application, vous devez configurer la connexion à Google Sheets :

  1. Créer un nouveau Google Sheets (vide). Le script créera automatiquement les feuilles nécessaires : Bandes, Saisies, Intrants, Matieres, FormulationsMP, Fournisseurs, Receptions, Inspections, Stocks, Clients, Commandes, LignesCommande
  2. Ouvrir Extensions > Apps Script, coller le code Code.gs (voir section Paramètres), déployer en Application Web
  3. Copier l'URL de déploiement et la coller dans Paramètres
